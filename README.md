- **Canvas WebGL animado** — Logo de JOM STUDIO con partículas reactivas al cursor
- **Cursor personalizado** con efectos magnéticos en botones
- **Lightbox avanzado** — Navegación entre fotos con teclado, swipe y fade
- **Photobook dinámico** — 4 sesiones fotográficas con galería modal
